Project description

The aim of this project is to detect political bias in any text.
To achieve this goal the input text is cleaned, tokenized and vectorized before being processed by the algorythm, that analyses the features of the text and assigns a category to it.

The categories used are based on the Project Manifesto Rile Index. The database used to extract said categories is the following:
Volkens, Andrea / Burst, Tobias / Krause, Werner / Lehmann, Pola / Matthieß Theres / Merz, Nicolas / Regel, Sven / Weßels, Bernhard / Zehnter, Lisa (2020): The Manifesto Data Collection.
Manifesto Project (MRG/CMP/MARPOR). Version 2020b. Berlin: Wissenschaftszentrum Berlin für Sozialforschung (WZB). https://doi.org/10.25522/manifesto.mpds.2020b
